# How to create a snippet

## Heading

You can create a heading like:

```ts
//# This is my heading

'...rest of your code'
```

## Description

You can create a description like:

```ts
/// This is my description

'...rest of your code'
```

## Demo

Your can be spllited by two section, if you create a demo comment 

```ts
// DEMO:
```
